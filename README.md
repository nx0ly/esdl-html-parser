# Haskell ESDL Project

## Overview
This project showcases a simple implementation of HTML generation using ESDL (Embedded Domain Specific Language) in Haskell. It provides a type-safe way to create HTML structures programmatically, leveraging Haskell's robust type system for reliable web content generation.

## Features
- Generates valid HTML5 markup
- Supports dynamic content creation
- Extensible structure for adding complex HTML elements

## Getting Started

### Prerequisites
- Haskell (GHC 8.10 or later)

### Installation
1. Clone the repository.

2. Navigate to the directory.

3. Run it
    ```bash
    runghc main.hs <input file> <output file>
    ```
